The Boozer Catalogue is a cocktail chooser website that uses the thecocktaildb.com API to give its users access to a large variety of cocktail recipes. They can search by category of drink from the homepage, or by name with the search bar. They also have the option to have a random cocktail generate from the entire database in the 'Dealers' Choice' section.

Snippet is a minimalistic app using the Reddit API, showing a selection of recent posts. Users can search via the searchbar at the top, or via the current top subreddits to the side. The first ten comments on any post can be shown or hidden, and users can be directed straight to the original post on Reddit if they wish.

Foreteller is a simple little app that randomly chooses a fortune-cookie style fortune at the click of a button.
